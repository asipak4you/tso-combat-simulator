# settlersonlinesimulator.com

Statische Web-App des **The Settlers Online Kampfsimulators** – ein exakter
analytischer Löser, der vollständig im Browser läuft (WebAssembly). Kein
Server-Compute.

**Live:** https://settlersonlinesimulator.com

Diese Dateien sind ein **Build-Artefakt**. Der Quellcode (Python-Referenz,
C-Port, Spezifikation, Tests, Build-Skripte) liegt im privaten Repository.
